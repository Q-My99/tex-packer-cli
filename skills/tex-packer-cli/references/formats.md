# Formats

## Exporters

Built-in exporters:

- `JSON (hash)`
- `JSON (array)`
- `XML`
- `css (modern)`
- `css (old)`
- `pixi.js`
- `Godot (atlas)`
- `Godot (tileset)`
- `Phaser (hash)`
- `Phaser (array)`
- `Phaser 3`
- `Spine`
- `cocos2d`
- `UnrealEngine`
- `Starling`
- `UIKit`
- `Unity3D`
- `Egret2D`
- `custom`

Custom templates receive `rects`, `config`, and `appInfo`.

## Splitters

Supported splitter metadata:

- `Grid`
- `JSON (hash)`
- `JSON (array)`
- `XML`
- `UIKit`
- `Spine`

## Filters

- `none`
- `mask`
- `grayscale`
